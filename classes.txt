Проектирование классов. Какие классы нужны:

1. Task - Создает задания, возвращает статусы заданий. Получает из БД информацию по заданиям.
2. Category - класс, который будет получать категории из БД
3. City - класс, который будет получать города из БД
4. User - класс, который обрабатывает информацию по пользователям. Создает новых пользователей с ролями, форматирует информацию из БД по пользователям, считает их рейтинг.
5. Reviews - класс формирует отзывы заказчиков
6. Responses - создает отклики работников на задачи
